# CoverFlow jQuery Plugin

iTunes-style coverflow using jQuery and CSS3

## Getting Started

Download the [production version][min] or the [development version][max].

[min]: https://raw.github.com/xerode/jquery-jquery-coverflow/master/dist/jquery.jquery-coverflow.min.js
[max]: https://raw.github.com/xerode/jquery-jquery-coverflow/master/dist/jquery.jquery-coverflow.js

In your web page:

```html
<script src="jquery.js"></script>
<script src="dist/jquery-coverflow.min.js"></script>
<script>
jQuery(function($) {
  $.awesome(); // "awesome"
});
</script>
```
